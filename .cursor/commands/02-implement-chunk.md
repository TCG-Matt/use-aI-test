# Command: Implement Next Chunk

**Reference:** See `spec.md` > Implementation Plan.

**Action:**
1.  Identify the next unchecked box in the Implementation Plan.
2.  **Context Check:** Read the relevant existing files that this new code will touch.
3.  **Test Plan:** Create (or update) a test file that will verify this specific chunk works.
4.  **Implement:** Write the code to satisfy the test and the requirement.

**Constraint:** Only do this ONE step. Do not jump ahead to future steps. Stop and ask for verification/testing after this step is done.
